Mis Habilidades

HTML

HTML es el lenguaje de marcado que utilizamos para diseñar sitios web. Es un lenguaje estructurado que permite crear páginas web de manera sencilla y fácilmente mantenible.

CSS

CSS es un lenguaje de hoja de estilo en cascada que se utiliza para dar estilo a los elementos del documento HTML. CSS permite definir propiedades y valores que controlan cómo se renderizan y aparecen los elementos en un navegador web.

JAVA SCRIPT

JavaScript es un lenguaje de programación interpretado y ejecutable en el navegador web. Es un lenguaje de alto nivel, interpretado por un intérprete de JavaScript, que se utiliza para crear páginas web interactivas y para realizar tareas en el lado del cliente.
